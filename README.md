#  Heart Disease Prediction App

A simple Machine Learning web application that predicts the risk of heart disease based on user inputs.

---

##  Project Overview

This project demonstrates a basic end-to-end ML workflow, from data preprocessing to model deployment using Streamlit.

---

##  Work Done

* Performed basic data preprocessing
* Applied one-hot encoding for categorical features
* Used StandardScaler for feature scaling
* Trained a Logistic Regression model
* Saved model and scaler using pickle
* Built a Streamlit web app for real-time prediction

---

##  Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

---

##  How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## Output

* Takes user input (age, BP, cholesterol, etc.)
* Predicts:

  *  Low Risk
  *  High Risk

---

##  Purpose

This project was built to understand and implement a basic Machine Learning pipeline and deploy it as a simple web application.
